# CreatorFlow AI

CreatorFlow AI is a modern AI-powered content generation platform designed to simplify digital content creation through dedicated AI workspaces. Unlike traditional chatbot interfaces, CreatorFlow AI provides specialized generators for **Text, Code, Images, and Audio**, allowing users to create high-quality content in a focused and efficient environment.

The platform is built with productivity, creativity, and user experience in mind. Users can generate AI-powered content, organize their work into projects, access intelligent prompt libraries, manage generation history, and share their work professionally using their own LinkedIn and GitHub accounts.

---

## Features

### Text Generator

Generate professional written content, including:

- Blog posts
- Professional emails
- Technical documentation
- Reports
- LinkedIn posts
- Marketing content
- Essays

### Code Generator

Generate production-ready code in multiple programming languages and frameworks, including:

- Java
- Spring Boot
- React
- TypeScript
- JavaScript
- Python
- SQL
- HTML
- CSS
- C#

### Image Generator

Generate unique AI-powered images from user prompts.

Features include:

- AI-generated images
- Generate new image variations
- Download generated images
- Save images to projects

### Audio Generator

Supports both Text-to-Speech (TTS) and Speech-to-Text (STT).

**Text-to-Speech**

- Convert written text into natural-sounding speech
- Play, pause, replay, stop, and download audio

**Speech-to-Text**

- Record audio
- Convert speech into editable text
- Edit the transcription before generating audio

### Prompt Library

Each generator includes its own dedicated Prompt Library.

Users can:

- Select professional prompt templates
- Refresh prompt suggestions
- Generate new prompt ideas specific to the selected generator

### Projects

Organize generated content into projects.

Projects can contain:

- Generated text
- Source code
- Images
- Audio
- Saved prompts

### History

Store previous generations and quickly:

- Reuse prompts
- Copy outputs
- Save to projects
- Delete history

### Professional Sharing

Share generated content using your own professional accounts.

Supported platforms include:

- LinkedIn
- GitHub
